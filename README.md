# 💍 Josh + Rain Wedding Website

A simple, elegant, and responsive wedding website built with HTML, CSS, and JavaScript.
Designed to share our special day with friends and family — with a clean layout, smooth interactions, and mobile-friendly experience.

---

## ✨ Features

* 📅 Event details (date, location, schedule)
* 👗 Dress code section with visual inspiration
* 📍 Interactive map modal
* 📩 RSVP system (Firebase integration ready)
* ❓ FAQ section with collapsible answers
* 📱 Fully responsive design (desktop + mobile optimized)
* 🎨 Handwritten-style aesthetic using Google Fonts

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Flexbox)**
* **JavaScript (Vanilla)**
* **Firebase** (for RSVP data handling)

---

## 📸 Sections Overview

* **Hero** – Intro banner with names and date
* **Schedule** – Timeline of the event
* **Location** – Map and venue details
* **Dress Code** – Outfit inspiration with modal preview
* **RSVP** – Guest response form
* **FAQ** – Common questions

## 🚀 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. Open the project folder:

   ```bash
   cd your-repo-name
   ```

3. Run locally:

   * Just open `index.html` in your browser

---

## 🔥 Firebase Setup (Optional)

To enable RSVP storage:

1. Create a Firebase project
2. Enable Firestore Database
3. Copy your Firebase config
4. Paste it into your JavaScript file
5. Connect it to your RSVP form logic
